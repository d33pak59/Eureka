# Eureka Server 

Eureka Server is a service registry used for service discovery in the Quiz Application microservices architecture. All microservices register themselves here.

##  Tech Stack
- Java
- Spring Boot
- Spring Cloud Netflix Eureka Server

##  Features
- Service Registration
- Service Discovery
- Centralized Registry for Microservices
